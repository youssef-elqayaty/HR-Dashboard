# HR Analytics Dashboard - Power BI

## Project Overview

This project analyzes employee attrition patterns and workforce demographics using Power BI.

The objective is to help HR teams and business leaders identify the factors contributing to employee turnover and support data-driven workforce decisions.

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling

## Key Metrics

* Total Employees
* Attrition Count
* Attrition Rate %
* Average Employee Age
* Average Years at Company

## Business Questions Answered

### Workforce Overview

* How many employees are currently in the organization?
* What is the average employee age?
* What is the average tenure?

### Attrition Analysis

* What is the overall attrition rate?
* Which departments have the highest employee turnover?
* Which job roles experience the highest attrition?

### Employee Demographics

* How is attrition distributed across age groups?
* What is the gender distribution of employees?

### Compensation Insights

* Is employee attrition associated with specific salary ranges?

### Experience Analysis

* How does total work experience relate to employee turnover?

## Dashboard Features

* Interactive filters (Department, Age Group)
* KPI cards
* Department-wise attrition analysis
* Salary-based attrition analysis
* Job role comparison
* Age group distribution
* Gender analysis
* Experience trend analysis
